1) 회원가입 · 학교 인증 — 신뢰 기반 커뮤니티의 출발점

사용자는 경북대 학번과 학교 이메일(@knu.ac.kr)로만 가입할 수 있으며, 학번·닉네임 중복 확인 절차를 거친다. 학교 이메일 인증을 통해 재학생 여부를 검증함으로써, 기존 익명 오픈채팅방의 한계였던 '책임지지 않는 익명성' 문제를 입구 단계에서부터 차단하고 신뢰 기반 커뮤니티의 토대를 마련한다.

2) 홈 피드 — 태그 기반 실시간 모임 탐색

3) 모임 만들기 — 조건 게시 즉시 매칭 시작

모임을 주최하려는 사용자는 제목, 태그, 장소, 만나는 날짜·시간, 모집 인원(2~10명), 간단한 소개, 그리고 카카오 오픈채팅 링크를 입력해 게시물을 등록한다. 게시 즉시 홈 피드에 노출되며 매칭이 시작된다. 이때 입력한 오픈채팅 링크는 화면에 바로 공개되지 않고 정원이 충족되어 매칭이 완료된 참여자에게만 공개되어, 무분별한 외부 유입을 막는다.

4) 모임 상세 · 참여 신청 — 실시간 선착순 정원 매칭
[스크린샷 ④ 모임 상세 화면 (참여 현황 바)]
관심 있는 모임을 누르면 상세 정보와 함께 실시간 참여 현황 게이지가 표시되고, '참여 신청하기' 버튼으로 즉시 신청할 수 있다. 다수가 동시에 신청해 정원이 마감되면 밀려난 사용자에게는 자동으로 '신청 실패(정원 마감)'가 처리되어, 별도 조율 없이 실시간 선착순 매칭이 이루어진다. (주최자 본인은 신청 불가, 신청 후 취소도 가능)
5) 매칭 완료 · 오픈채팅 자동 연결 — 대기 없는 즉시 연결
[스크린샷 ⑤ 알림 화면 / 매칭완료 + 오픈채팅 버튼]
정원이 충족되면 모임이 '매칭완료' 상태로 전환되고, 참여자에게는 매칭 완료 알림(🎉)과 함께 오픈채팅 참여 버튼이 발송된다. 사용자는 별도의 대기나 수동 연락 과정 없이 버튼 한 번으로 바로 채팅방에 입장해 약속을 확정할 수 있어, 즉흥 모임 특유의 '신속한 연결'이라는 핵심 가치를 구현한다.
